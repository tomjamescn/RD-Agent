用户界面¶

介绍¶

RD-Agent 在研发过程中会生成一些日志。这些日志对于调试和理解研发过程非常有用。然而，仅查看终端日志并不够直观。RD-Agent 提供了一个网页应用作为用户界面，以可视化研发过程。您可以轻松查看研发过程，更好地理解研发过程。

快速演示¶

启动网页应用¶

在 RD-Agent/ 文件夹中，运行：

rdagent ui --port <port> --log_dir <log_dir like "log/"> [--debug]

这将在 http://localhost:<port> 启动一个网页应用。

注意：log_dir 参数不是必需的。您可以在网页应用中手动输入 log_path。如果您设置了 log_dir 参数，您可以在网页应用中轻松选择不同的 log_path。

--debug 是可选的，它将在侧边栏中显示一个 "单步运行" 按钮和网页应用中的已保存对象信息。

使用网页应用¶

  1. 打开侧边栏。

  1. 选择您想要展示的场景。有一些预定义的场景：

    • Qlib 模型

    • Qlib 因子

    • 数据挖掘

    • 论文中的模型

    • Kaggle

  2. 点击 配置⚙️ 按钮并输入日志路径（如果您设置了 log_dir 参数，您可以在下拉列表中选择一个 log_path）。

  3. 点击配置⚙️下方的按钮以显示场景执行过程。按钮包括：

    • 所有循环：显示完整的场景执行过程。

    • 下一个循环：显示一个成功的 研发循环。

    • 一个演变：显示一个**演变**的**发展**步骤。

    • 刷新日志：清除显示的日志。
